Modals

A lightweight, accessible modal dialog component.

Modal Sizes

Small Modal

This is a small modal dialog.

Default Modal

This is a default sized modal dialog with some content.

You can add more content here as needed for your application.

Large Modal

This is a large modal dialog with plenty of space for content.

You can add much more content here for complex forms or detailed information.

The modal is also fully responsive and will adapt to different screen sizes.

<!-- Button to open the modal -->
<button class="sleek-btn sleek-btn-primary" data-sleek-modal-open="myModal">Open Modal</button>

<!-- Modal component -->
<div class="sleek-modal sleek-modal-md" id="myModal">
  <div class="sleek-modal-content">
    <div class="sleek-modal-header">
      <h4 class="sleek-modal-title">Modal Title</h4>
      <button class="sleek-modal-close" data-sleek-modal-close aria-label="Close">
        <!-- Close icon -->
      </button>
    </div>
    <p>Modal content goes here...</p>
    <div class="sleek-modal-footer">
      <button class="sleek-btn sleek-btn-ghost" data-sleek-modal-close>Cancel</button>
      <button class="sleek-btn sleek-btn-primary">Confirm</button>
    </div>
  </div>
</div>
